# Meraki-Dashboard-TV-NOC-Mode-Local-Host-Web-Portal
A lightweight, dark-mode Flask web application designed specifically for Network Operations Centers (NOCs) and wall-mounted TV displays. This application connects to the Cisco Meraki API to provide a real-time, high-visibility dashboard of your network infrastructure, builtin with MFA.

Compiler EXE
pyinstaller --onefile  --name "Meraki_TV_Monitor" script.py
